① ググラボの使い方（基本操作）

ログイン方法

  1. ブラウザで本サービスのURLを開く
  2. 登録済みのメールアドレスとパスワードを入力
  3. 「ログイン」ボタンをクリック

ダッシュボードの見方

ログイン後に表示されるトップページがダッシュボードです。

  • QRコード生成：店舗のGoogleレビューQRコードを作成できます
  • AI返答生成：お客様のレビューに対する返答文をAIが自動作成します
  • アンケート設定：お客様向けアンケートの内容（サロンタイプ・メニュー選択肢など）を設定できます
  • アンケート回答一覧：お客様が回答したアンケートを時系列で確認できます。AI生成口コミの全文表示にも対応しています

店舗の登録方法

  1. ナビゲーションの「QRコード生成」をクリック
  2. 「店舗名」「スラッグ（URL用の英数字）」「GoogleレビューURL」「Place ID」を入力
    ※ Place IDはLINE連携に必要な項目です（必須）。取得方法は⑥ Place IDの取得方法を参照してください
  3. 「店舗を追加」ボタンをクリックして登録完了
  4. 登録した店舗を選択してQRコードを生成できます

印刷サイズの目安

③ お客様にレビューを書いてもらうための工夫

  • 1
    QRコードの横に一言添える
    「口コミを書いてくれると嬉しいです」などのメッセージをPOPに添えると、行動を促すきっかけになります。
  • 2
    会計時にスタッフが一言声がけをする
    「よろしければQRコードからご感想をいただけると嬉しいです」と自然に伝えると効果的です。
  • 3
    星をタップするだけで書けることを伝える
    「文章を書かなくても星だけでもOKです」と伝えると、レビューのハードルが下がります。
  • 4
    レビューを書いてくれたお客様にクーポンを渡す
    「レビューを書いていただいた方に次回使えるクーポンをプレゼント」とすると投稿率が上がります。本サービスのクーポン機能をご活用ください。

⑥ アンケート設定の使い方

アンケートはQRコードを読み取ったお客様が回答するページです。サロンの業態に合わせて内容を最適化できます。

サロンタイプの選択

ナビゲーションの「アンケート設定」から店舗ごとにサロンタイプを選択してください。
選択したタイプに応じて、Q1（メニュー選択）・Q3（よかった点）の選択肢が自動で切り替わります。

💇 美容院（一般）
💈 メンズサロン
🌸 女性専用サロン
💆 ヘッドスパ専門
✨ 縮毛矯正専門

アンケートの質問構成

  • Q1 ご利用メニュー：ボタン選択（サロンタイプ別の選択肢）
  • Q2 満足度：⭐〜⭐⭐⭐⭐⭐の5段階評価
  • Q3 よかった点：複数選択可（サロンタイプ別の選択肢）
  • Q4 また来たいですか？：はい／考え中／まだわからない
  • Q5 メッセージ：自由記述（任意）

AI口コミ生成について

お客様がアンケートを送信すると、回答内容をもとにAIがGoogleレビュー用の口コミ文を自動生成します。
生成された口コミはアンケート画面でお客様に表示され、コピーしてそのままGoogleに貼り付けられます。
サロンタイプごとに文体・強調ポイントを最適化しています（例：女性専用サロンはプライベート感を重視）。

⑦ アンケート回答ダッシュボードの見方

ダッシュボード下部の「アンケート回答一覧」でお客様の回答を確認できます。

表示される項目

  • 受付日時：アンケート送信日時
  • メニュー（Q1）：お客様が選んだメニュー
  • 満足度（Q2）：星の数（1〜5）
  • よかった点（Q3）：選択されたポイント
  • 再来店意向（Q4）：はい／考え中／まだわからない
  • お客様のメッセージ（Q5）：自由記述の内容
  • AI生成口コミ：「全文を見る」ボタンでモーダル表示。コピーボタンも利用可

フィルター機能

複数店舗をお持ちの場合は、店舗名で絞り込むことができます。
フィードバックの傾向をつかみ、サービス改善にお役立てください。

⑧ LINE連携の設定方法

LINE連携を設定すると、週次のGoogleレビューレポートがLINEに届きます。

LINE連携には店舗登録時に Place ID の入力が必要です

連携手順

  1. 「QRコード生成」ページで店舗カードを確認する
  2. 「LINE連携が未設定です」のバナーが表示されている場合は「LINEを連携する」をクリック
  3. LINEの友だち追加画面が開くので、ugugulabo公式アカウントを友だち追加する
  4. 連携完了後、毎週月曜日に先週のレビューサマリーがLINEで届きます

Place IDの取得方法

  1. PCのブラウザで developers.google.com/maps/documentation/places/web-service/place-id を開く
  2. ページ下部の「Place ID Finder」で自分の店舗名を検索する
  3. マップに表示されたマーカーをクリックすると Place ID が表示される
  4. 表示された ChIJ... から始まる文字列をコピーして店舗登録フォームに入力する

⑨ GoogleレビューURLの取得方法

方法1または方法2のPC操作が最も確実です
方法1 - 推奨 Google ビジネスプロフィールから取得（PC）
  1. PCのブラウザで business.google.com を開く
  2. 自分の店舗を選択する
  3. 「レビューを増やす」→「リンクを取得」をクリック
  4. 表示されたURLをコピーしてググラボに貼り付ける
方法2 Googleマップから取得（PC）
  1. PCのブラウザで maps.google.com を開く
  2. 自分の店舗名を検索
  3. 「レビューを書く」ボタンを右クリック →「リンクのアドレスをコピー」
  4. コピーしたURLをググラボに貼り付ける
方法3 Googleマップアプリから取得（スマホ）
  1. Googleマップアプリを開く
  2. 自分の店舗を検索してタップ
  3. 「共有」ボタンをタップ →「リンクをコピー」
  4. コピーしたURLをググラボに貼り付ける

※ スマホの場合は店舗ページのURLになります
